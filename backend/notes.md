- Activar el entorno virtual:

```bash
.\env\Scripts\activate
```

- Crear proyecto:

```bash
django-admin startproject backend .
```

- Crear rama:

```bash
git branch nombreRama
```

- Cambiar de rama:

```bash
git switch nombreRama
```

- Traer cambios entre ramas:
```bash
git merge nombreRama 
```
